## Features

## Shadcn UI

Compatible with [shadcn/ui](https://ui.shadcn.com/) components, allowing you to use their pre-built components in your documentation.

import { Accordion, Accordions } from 'fumadocs-ui/components/accordion';

<Accordions>
  <Accordion title="Hy?">
    Hye
  </Accordion>
</Accordions>
