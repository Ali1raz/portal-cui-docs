## What is fumadocs

Fumadocs is a React.js documentation framework for Developers, beautifully designed by Fuma Nama. 
Bringing powerful features for your docs workflows, 
with high customizability to fit your preferences, works seamlessly with any React.js framework, CMS — anything.

## Installation

```bash
git clone fumadocs-starter.git
cd fumadocs-starter
```

## Install dependencies

```npm
npm install
```

## Run the development server

```npm
npm run dev
```
